Student Registration System

A Student Registration System is a simple yet effective example of a To Do List project, This system allows users to add, Edit, and Delete student information in a structured way.
Adding Student Details: Users can input essential student information such as name, age, and class, similar to adding tasks in a To-Do list.
Editing/Deleting Student Information: Users can modify existing student details or delete records if necessary, mirroring how tasks are edited or removed in a To-Do list.
List View: Just like a task list, all students are displayed in a list format, making it easy to view and manage multiple records at once
